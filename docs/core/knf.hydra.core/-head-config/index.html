Content copied to clipboard

Module configuration, enable and modify the behaviour of some features.

To use this class you need to create a new class and extend HeadConfig or use a kotlin object, then modify the required variables in init, for example:

object: HeadConfig() {
   init {
      isRecentsAvailable = true
      isSearchAvailable = true
      ...
   }
}
